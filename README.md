[VS Code](https://code.visualstudio.com/) + [Vue (Official)](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (and disable Vetur).

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

## Email setup (Resend)

1. Copy `.env.example` to `.env`.
2. Fill these values in `.env`:
   - `RESEND_API_KEY`
   - `RESEND_FROM_EMAIL` (must be verified in Resend)
   - `RESEND_TO_EMAIL` (where farmer form messages are delivered)
3. Start frontend + backend together:

```sh
npm run dev:full
```

The farmer form sends requests to `/api/send-email`, which is handled by `server/index.js`.
